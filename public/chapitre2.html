Chapitre 2


Synchronisation des dépots

Récuperation du code du dépot distant: git pull

Envoie du code local vers le dépot distant: git push

Créer une autre branche:

Git checkout -b nomdelabranche
-b permet de créer une branche. Pour le moment elle est crée en local il faut ensuite faire

git push -u origin nomdelabranche

Pour se balader de branche en branche il suffit de faire :

git checkout nomdelabranche
